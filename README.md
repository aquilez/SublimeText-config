SublimeText 3
===================

This is the config I use for SublimeText 3.

Packages included:

* Package Control
* Alignment
* CleanCSS
* CSS3 Syntax
* DocBlockr
* Emmet
* SideBarEnhancements
* Laravel Blade Highlighter

Themes/Color Schemes included:

* Spacegray

Notes
===================

* Many lang packages have been disabled
* Default line ending is unix-style
* TABs rule in this config
